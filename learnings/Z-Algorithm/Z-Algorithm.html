Search the substring

One of the most well known use case of Z-box algorithm involves string matching. The question states, given two strings, one a string of text namely t and pattern p, find all occurrences of the pattern p inside the text t. 

Example:

T Text   : aabcaabxaaz
P Pattern: bcaa

The primitive operation here is that of character comparison. What is primitive operation? The operation which forms the basis of finding time complexity for an algorithm is a primitive operation. Classic example is number of swaps while sorting, say in bubble sort. We say the average case time complexity of bubble sort is O(N2)O(N^2)O(N2)﻿ because on average that is how many swaps we see while sorting the array using bubble sort.

Obvious solution:

Align the pattern with the first character of the text. Now try to match the pattern character by character as much as you can. If you are over with the pattern well and good! If not however, align the pattern with the second character in the text and now start matching character by character both the pieces of text. Also, here since we need to find all the occurrences of the pattern, we anyways will be performing the matching step from each character of the starting text. The time complexity here will hence be O(∣P∣∗∣T∣)O(|P|*|T|)O(∣P∣∗∣T∣)﻿.

Time Analysis

Claim: The algorithm does at most O(∣S∣)O(|S|)O(∣S∣)﻿ character comparisons, so the time taken by the algorithm is O(∣S∣)O(|S|)O(∣S∣)﻿.

Proof: The variable rrr﻿, never decreases. In any iteration rrr﻿ stays the same or moves to the right by an amount at least as large as the number of matches   ⟹  \implies⟹﻿The number of matches ≤∣S∣\leq |S|≤∣S∣﻿. Similar to the bouncing ball proofs where we look at a variable and see a ball bouncing over it to count how many times the ball moves and how the ball moves. Also, in any iteration where some comparisons are done, the iteration ends when the first mismatch occurs   ⟹  \implies⟹﻿the number of mismatches ≤\leq≤﻿ ∣S∣|S|∣S∣﻿. Hence the number of comparisons ≤2∣S∣=O(∣S∣)\leq 2|S| = O(|S|)≤2∣S∣=O(∣S∣)﻿

Implementation [In Python3]

def getZBoxValues(str):
    zBox = [0 for character in str]
    zBox[0] = len(str)
    l, r, k = 1, 1, 1
    while k < len(str):
        if k >= r:
            l, r = k, k
            while r < len(str) and str[r] == str[r - k]:
                r += 1
            zBox[k] = r - l
        elif k < r:
            betaLength = r - k
            if zBox[k - l] < betaLength:
                zBox[k] = zBox[k - l]
            elif zBox[k - l] > betaLength:
                zBox[k] = betaLength
            else:
                l = k
                while r < len(str) and str[r] == str[r - k]:
                    r += 1
                zBox[k] = r - l
        k += 1
    return zBox


if __name__ == "__main__":
    text = "aabcaabcaaz"
    pattern = "bcaa"
    concatenated = pattern + text
    zBox = getZBoxValues(concatenated)
    print("The Z-Box -> %s" % zBox)
    for index, zBoxValue in enumerate(zBox):
        if index >= len(pattern) and zBoxValue >= len(pattern):
            print("Pattern found at location -> %s" % index)
